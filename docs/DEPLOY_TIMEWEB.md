# Деплой на Timeweb

## База данных PostgreSQL

1. В панели Timeweb создайте базу PostgreSQL (если ещё нет).
2. Скопируйте хост, имя БД, пользователя и пароль.
3. В `.env` укажите:

```
DB_HOST=ваш-хост.twc1.net
DB_NAME=имя_базы
DB_USER=пользователь
DB_PASSWORD=пароль
DB_PORT=5432
```

При подключении к удалённому хосту `*.twc1.net` SSL включается автоматически (`sslmode=require`).

## Медиафайлы

По умолчанию скриншоты и видео из отчётов сохраняются **локально** в папку `media/`.  
При редеплое локальные файлы могут пропадать — для постоянного хранения настройте S3 (см. `docs/MEDIA_S3_SETUP.md`).

## Переменные окружения

См. `env.example` — скопируйте в `.env` и заполните для продакшена.
